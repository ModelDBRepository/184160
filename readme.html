Notes

This model was converted by the ModelDB team from a model in the Allen Cell Types Database of a Rorb-IRES2-Cre-D neuron from layer 5 of the mouse primary visual cortex.

Their model was based on a traced morphology after filling the cell with biocytin and optimized using experimental electrophysiology data recorded from the same cell. The electrophysiology data was collected in a highly standardized way to facilitate comparison across all cells in the database. The model was optimized by a genetic algorithm that adjusted the densities of conductances placed at the soma to match experimentally-measured features of action potential firing.

Data and models from the Allen Cell Types Database are made available to the community under the Allen Institute's Term of Use and Citation Policy.

Reference

1. Allen Institute (2015) Documentation Allen Cell Types Database.
